# 🤖 Crypto Autopilot — автономный спот-трейдер на Bybit

Самостоятельный торговый бот, который работает **24/7 без твоего компьютера**: торгует
спотом по грид-стратегии (продал дороже — откупил дешевле), сам защищается от обвалов,
следит за новостями и отвечает тебе прямо в Telegram-боте.

Собран так, что даже полная утечка ключей не даст никому вывести твои деньги (API-ключ
создаётся **без права вывода**), а плечо/шорты отключены на уровне правил.

## Как это устроено — 3 контура

| Контур | Где живёт | Что делает |
|--------|-----------|-----------|
| 🏦 **Биржа** | Bybit | Держит лимитные заявки (тейки/откупы) — исполняются сами 24/7 |
| ⚙️ **Механика** | твой сервер (cron) | Каждые 30 мин: грид-цикл, отчёты, алерты. Простой Python без ИИ |
| 🧠 **Мозг** | GitHub Actions | Каждый час: анализ рынка/новостей, решения. Это Claude Code |
| 📱 **Бот** | Telegram | Отчёты, команды (докупи/продай/стоп), живой чат-вопросы |

**Почему нужен сервер?** Bybit блокирует IP дата-центров GitHub (США), поэтому все запросы
к бирже идут через твой сервер по SSH. Сервер — «руки», GitHub — «голова».

## Что понадобится

- Аккаунт **Bybit** + API-ключ (Trade, **без Withdraw**)
- **Telegram-бот** (создаётся у @BotFather за минуту)
- **Сервер** с Python 3.10+, SSH и cron (подойдёт дешёвый хостинг/VPS, ~$2-5/мес)
- Аккаунт **GitHub** (бесплатных Actions-минут хватает с запасом)
- Подписка **Claude** (Pro/Max) — мозг работает на ней, не на платном API

## Установка

👉 **Полная пошаговая инструкция — в [SETUP.md](SETUP.md).**

**Быстрый путь (рекомендую):** склонируй репо, открой папку в **Claude Code** и скажи:
*«Разверни этого бота под мой сервер и мою биржу, проведи меня по шагам»*. Именно так был
собран оригинал — ассистент сам настроит SSH, cron, вебхук и секреты, а тебя попросит
только создать ключи и подтвердить действия.

## Другая биржа

Код завязан на Bybit V5 API (`autopilot/bybit_client.py`). Для другой биржи (Binance, OKX…)
нужно переписать этот модуль под её API — остальная логика (грид, индикаторы, отчёты)
переносится. Claude Code это сделает, если попросить.

## Безопасность

- API-ключ — только Trade, **никогда не давай Withdraw**
- Бот отвечает только владельцу (твой `chat_id`), чужие сообщения игнорирует
- Секреты — в GitHub Secrets и `.env`/`config.php` (в `.gitignore`), не в коде
- Спот, без плеча — максимальный убыток ограничен вложенным

⚠️ Это не финансовый совет и не гарантия прибыли. Крипта рискованна. Торгуй на сумму,
которую готов потерять. Автор/код ответственности за убытки не несут.
